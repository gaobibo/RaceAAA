# RaceAAA
RaceAAA is a race and event management system. In this project, JavaScript and bootstrap4 are used for client-side programming; On the server-side, the flask-sqlalchemy and mysql-connector-python packages are used for database programming. In both runner and event search, in order to avoid the SQL injection, Python Regular Expression API is used to split userâ€™s input content to alphabet and number only words, and SQL Regular Expression operator is used with the predefined pattern to find out the matching records.
